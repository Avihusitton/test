# גיל סיטון — מטפלת רגשית | M.S.W

אתר הבית המקצועי של גיל סיטון, עובדת סוציאלית קלינית. האתר מציג את השירותים הטיפוליים לילדים ולמבוגרים, הגישה הטיפולית ופרטי התקשרות.

## מבנה הפרויקט

- `index.html` - עמוד הבית
- `about.html` - אודות וניסיון מקצועי
- `children-therapy.html` - טיפול רגשי לילדים
- `adult-therapy.html` - טיפול רגשי למבוגרים
- `contact.html` - יצירת קשר
- `assets/` - משאבי האתר (CSS, JS, תמונות)

## טכנולוגיות

- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- Responsive Design
- SEO Optimized
